# Personal Introduction Program 

## Project Description

This is a beginner Python program that asks the user for their **name, age, and favorite hobby**.
The program then displays a friendly **personalized welcome message** using the information provided by the user.

This project helps beginners understand how to use:

* `input()` to get user information
* Variables to store data
* `print()` to display output
* Basic Python program structure

---

##  Technologies Used

* Python 3

---

##  How to Run the Program

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the project folder in your terminal.
4. Run the program using:

```bash
python personal_intro.py
```
